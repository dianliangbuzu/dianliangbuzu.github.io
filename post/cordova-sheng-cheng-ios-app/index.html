Cordova - 打包IOS APP

2020-08-27 cordova

前言

公司使用vue开发的h5应用, 目前已经上线了Android端。
由于市场的需要公众号端已经没法满足用户的需求。
了解现在前端打包的方案。但是最终还是选择了Cordova(当然是因为以前用过🐝)

准备

1.NPM安装Cordova

npm install -g cordova

2.创建项目

cordova create MyApp

3.添加平台(ios)

cd MyApp
cordova platform add ios

项目目录结构

MyApp/
|-- hooks/ 脚本代码, 用来自定义 Cordova CLI 命令
|-- platforms/  平台源码
| |-- ios/
|-- plugins/ 提供javascript接口能够访问例如照相机和电池状态相关的事项。
  | |--cordova-plugin-whitelist/
  | |--browser.json
  | |--fetch.json
  | |--ios.json
|--  res/
  | |--icon/
  | |--screen/
|-- www/  HTML, JavaScript 和 CSS 文件
|-- config.xml 在 config.xml 中定义的东西应该适应所有平台
